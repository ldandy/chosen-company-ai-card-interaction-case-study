# Chosen Company Cards Project Bundle

This bundle collects the downloadable outputs and working deliverables created during the Chosen Company Cards AI-assisted interaction design case study.

## Project summary

Chosen Company Cards started as a family-friendly conversation deck for after-dinner gatherings, games nights, and extended-family use. It became a workflow case study showing how AI can help turn a rough idea into a structured interaction system people can touch, test, change, and use.

The bundle includes source card copy, print-ready PDFs, visual concepts, mockups, workflow case study files, and the reusable Card-Based Interaction Designer Skill. It is best read as a draft AI workflow example, not as a polished game product.

## Folder contents

### 01-card-source

- `family_conversation_cards.md`  
  Source Markdown file for the 100-card deck. This is the editable source copy that can be reused for future template automation or content editing.

### 02-print-ready

- `chosen_company_cards_printable_A4.pdf`  
  Multi-page A4 printable card file. Designed for printing card fronts first, then flipping the paper and printing backs.

- `chosen_company_box_template_A4.pdf`  
  A4 printable box artwork and template for the card set.

### 03-visual-assets

- `family_fun_around_the_game_table.png`

- `case_study_time_analysis.png`  
  Time analysis image used in the case study.

  Additional generated image copies from the image workflow, included as reference backups.

### 04-case-study

- `chosen_company_ai_workflow_case_study.docx`  
  Editable Word document case study.

- `chosen_company_ai_workflow_case_study.pdf`

### 05-skill

- `card-based-interaction-designer-skill.zip`  
  Packaged Skill ZIP for upload or sharing.

- `card-based-interaction-designer-source/`  
  Source folder for the Skill, including `SKILL.md`, references, templates, and scripts.

### 06-preview-images

- Print preview PNGs for checking card front/back pages and box artwork before printing.

## Suggested use order

1. Review `family_conversation_cards.md` if you want to edit the card copy.
2. Print `chosen_company_cards_printable_A4.pdf` at 100 percent scaling.
3. Print fronts first, then reload paper and print backs using the matching back pages.
4. Print and assemble `chosen_company_box_template_A4.pdf` if using the box design.
5. Use the Word case study for sharing or adapting the AI workflow example.
6. Upload or inspect the Skill ZIP if you want to reuse the generic card-based interaction workflow.

## Notes

- The print-ready files are practical prototypes, not commercial prepress files.
- Test-print one front/back sheet before printing the full deck.
- The Skill and source files are included for reuse, review, and future adaptation.
- Some image assets are concept renders. They may need design polishing before public or commercial use.
- The format is not really the point. The point is the interaction it creates.
