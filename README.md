# Chosen Company Cards: AI Card Interaction Case Study

This is a draft, in-progress AI-assisted card-based interaction design case study and prototype.

The original use case was a family conversation card deck called **Chosen Company Cards**: something for after dinner that is fun enough for teenagers, safe enough for mixed generations, and meaningful enough to create real connection without making the room awkward.

The bigger point is the workflow. AI helped move a rough idea into something people can touch, test, change, and use: cards, categories, visual concepts, printable sheets, box artwork, mockups, a draft case study, and a reusable Skill for creating other card-based interaction systems.

The format is not really the point. The point is the interaction it creates.

## Status

Personal, experimental, draft work. Not polished. Not final. Not a finished commercial product or formal workplace case study. Shared as a practical example others can inspect, adapt, improve, and use.

## Suggested use

Use this as a starting point for:

- family games night
- team culture prompts
- learning reinforcement
- workshop activities
- change-management conversations
- AI workflow demonstrations
- safe conversation design experiments

Not every workplace problem needs to become a card game. But card-based prompts can be useful when people need a safer way to practise judgement, explore trade-offs, talk through change, or make a vague topic more concrete.

## Safety note

Card-based interaction tools are simple, but they can still create pressure if used poorly. For workplace use, protect people first.

- Make passing normal.
- Avoid forced disclosure.
- Avoid public ranking of people.
- Avoid questions that create blame.
- Keep power dynamics in mind.
- Do not use playful activities to collect sensitive personal information without clear consent and purpose.
