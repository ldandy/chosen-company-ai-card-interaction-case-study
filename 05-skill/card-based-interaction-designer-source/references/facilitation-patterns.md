# Facilitation Patterns

## Self-guided mode

Use for low-risk family, social, or learning decks.
Requirements:
- Short rules.
- Pass option.
- Clear prompts.
- No sensitive capture.

## Lightly facilitated mode

Use when a group benefits from pacing or shared rules.
Requirements:
- One person reads the rules.
- Keep answers brief.
- Stop unkind teasing.
- Use a closing question if the activity had a purpose.

## Strongly facilitated mode

Use for workplace change, conflict-adjacent topics, mixed authority, leadership, sensitive culture work, or any group where answers could affect trust.
Requirements:
- Set group agreements.
- Make passing normal.
- Use scenario-based prompts.
- Avoid named criticism.
- Capture themes, not personal disclosures.
- Close with practical next steps.

## Anonymous mode

Use when insight is useful but named answers are risky.
Methods:
- People write answers without names.
- Use voting dots or sorting.
- Capture themes only.
- Do not imply anonymity if handwriting, context, or small group size makes people identifiable.

## Private reflection mode

Use for personal growth, coaching, or sensitive readiness.
Methods:
- Answer privately first.
- Invite voluntary sharing.
- Offer a lighter version of each prompt.

## Debrief questions

Choose 2 or 3:

- What did we notice?
- What surprised us?
- What did we agree on?
- What did we disagree on safely?
- What needs more clarity?
- What support would help?
- What is one small action from this?
- What should we test next?

## Pilot-test checklist

Ask after a small test:

- Were cards understood in under 10 seconds?
- Did anyone feel put on the spot?
- Were any cards boring, repetitive, or unclear?
- Did any cards trigger unhelpful teasing or defensiveness?
- Did the cards create useful conversation?
- Did the deck need more fun, more depth, or more structure?
- Which cards should be removed?
- Which cards should be rewritten?
